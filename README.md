# distro-customizer

Auto install all importent apps and customization for distro

## Available distros

- Fedora - testing..
- dabian - upoming..
- Arch - upcoming..

## How to run

- install git through packet manager

```
sudo apt install git
OR
sudo dnf install git
OR
sudo pacman -S git
```

- Clone this repo and cd into repo
- Run installer

```
sh install.sh
```
